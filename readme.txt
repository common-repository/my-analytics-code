My Analytics Code Wordpress is a simple plugin to include Analytics tracking.

== Description ==

My Analytics Code is a simple plugin to include Analytics tracking. After installing the plugin, just go to *Settings > My Analytics Code* and input your Analytics Tracking Code.

== Installation ==

1. Upload `my-analytics-code` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to *Settings > My Analytics Code* to configure.
